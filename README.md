# Ubi-Voyance

A simulation pipeline for blood vessel analysis.
Integrates SimVascular CFD, MCX Monte Carlo photon transport, and ML-based optical biomarker extraction for haemoglobin and white blood cell quantification.

## Stack
- Frontend: React
- Backend: Spring Boot
- Simulation: Python (pmcx, trimesh, VTK)
- ML: scikit-learn / tensorflow
- Database: PostgreSQL
- Hosting: Railway
